# Telecom Customer Churn Prediction

A machine learning project to predict customer churn using the Telco Customer Churn dataset.

## Models Used
Logistic Regression, SVM, KNN, Decision Tree, Naive Bayes, Random Forest, AdaBoost, XGBoost, MLP, SGD

## Best Model
Logistic Regression — F1-Score: 0.6092, Accuracy: 80.70%

## Dataset
IBM Telco Customer Churn Dataset (7,043 records, 20 features)
